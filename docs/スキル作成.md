# スキル作成ガイドライン

本ドキュメントは、AIエージェントが実行する「スキル（指示書）」を定義・更新する際の標準的な指針を定める。

## 1. 記述表現の厳格化

### 1.1. 命令形の徹底
スキル内の指示や手順には、「〜を推奨する」「〜が望ましい」といった選択の余地を与える曖昧な表現を一切使用してはならない。
AIエージェントが迷わず、一貫した動作を保証するため、**必ず「〜せよ」「〜してください」「〜を必須とする」といった強い命令形（Imperative）を使用すること。**

- **不可**: `ログを保存することを推奨します。`
- **可**: `ログを必ず .gemini/tmp/ 配下に保存してください。`
